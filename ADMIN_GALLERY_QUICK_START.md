# 🎨 Admin Gallery - Quick Start Guide

**One-Page Quick Reference for Using the New Gallery Manager**

---

## 🚀 Launch the Gallery

```
📍 Navigate to: /admin/gallery
🔑 You need: Supabase authentication (admin login)
⏱️  Load time: ~2 seconds
```

## 📤 Upload Images

**Option 1: Drag & Drop** (Easiest)
```
1. Find image on your computer
2. Drag it onto the gray upload area
3. Done! Image uploads automatically
```

**Option 2: Click to Select**
```
1. Click the upload area
2. Select file(s) from dialog
3. Click "Select Files" button
4. Uploads automatically
```

---

## 🖼️ Manage Your Images

### Copy URL (for code)
```
1. Hover over image
2. Click "Copy" button
3. Paste into code/database
4. Done!
```

### Delete Image
```
1. Hover over image
2. Click trash icon
3. Confirm deletion
4. Image removed
```

---

## 📏 Image Sizes

**Quick Reference:**

| Use | Size | Format |
|-----|------|--------|
| Space card | 500x400 | JPG |
| Space detail | 800x600 | JPG |
| Page hero | 1200x900 | JPG |
| Blog header | 800x400 | JPG |
| Avatar | 100x100 | JPG/PNG |

**Compression:**
- Use TinyPNG before uploading
- Aim for 200-500KB per image
- Max file size: 5MB

## ✅ Common Tasks

### "I want to update The Bar space photos"
```
1. Click "Studio Spaces" bucket
2. Click "the-bar" folder
3. Delete old images
4. Upload new images
5. Copy URLs if needed for code
```

### "I want to add a new page hero"
```
1. Click "Page Heroes" bucket
2. Click "pages/home" (or other page)
3. Delete old hero (if exists)
4. Upload new hero image
5. Refresh website to see changes
```

### "I want to add testimonial avatar"
```
1. Click "About & Testimonials" bucket
2. Click "testimonials" folder
3. Upload new avatar (100x100 recommended)
4. Name it same as creator
```

### "I want to add to portfolio gallery"
```
1. Click "Portfolio & Shoots" bucket
2. Upload images (no folder selection)
3. Images appear on /gallery page
4. Done!
```

## 🐛 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Upload fails | Check: file < 5MB, JPG/PNG/GIF format |
| Can't see images | Refresh browser, clear cache |
| Image not in folder | Re-upload, confirm folder selected |
| Delete didn't work | Try again, check permissions |
